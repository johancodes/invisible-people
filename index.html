Demo: Become Invisible

Try this out using your webcam. Click on button below and wait a few seconds for the model to load. Stand about three feet away from your webcam and start walking around in front of the camera. Watch as you rapidly disappear from the live video feed.

Note that non-human objects will still be seen moving in the background as normal. Best used in well-lit environments. Best experienced on a desktop computer. This demo has been repurposed by Joel Lim based on code written by Jason Mayes who produced this using Google's TensorFlow.js machine learning cloud platform.
